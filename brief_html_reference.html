Basic CSS

ways of styling elements

  1. Inline style using style statement
  2. Decorate multiple elements using html tags
  3. Creating particular class of decoration using CSS class
  4. Decorate particular id
  5. Through attribute selection

CSS treats each HTML element as its own box which is reffered to as the CSS Box Model. Block level items such as headings, paragraphs, divs etc start on a new line. Inline elements such as images and spans etc sit around other elements. Order of class declarations in the style section of the head tag is important because second declaration takes precedence over the earlier one and overrides its style. Class declaration overrides the body tag style declaration. Similarly the id declaration takes precedence over the class declaration. In this case it does not matter in which order the declarations appear in the head tag. The id declaration may appear early or late in the flow of the declaration It will always take precedence over the class declaration. Inline styling can take precedence over any other declaration in the style section of the document. In order to make sure that some style appear even in the case of accidental override of the style one should use a keyword called "!important" beside the style definition. e.g. color: red!important;

Features and values

  • text-align:justify, center, right, left
  • height:20px;
  • This is a 'strong' which makes the text bold
  • 'u' tag adds underline, 'em' tag creates underline, 's' tag creates strike-through, 'hr' tag creates horizontal line
  • box-shadow:offset-x offset-y blur-radius spread-radius color (blur-radius and spread-radius are optional entries). Multiple bos shadows can be created using commas. e.g. box-shadow: 0 10px 20px rgba(0,0,0,0.19), 0 6px 6px rgba(0,0,0,0.23);
  • Opacity: 0 to 1; 1 is fully opaque and 0 value is fully transparent. it makes an element opaque or transppaprent.
  • To transform texts in order to make it uppercase or lowercase or capitalize, initialize etc we use text-transform. e.g. text-transform: lowercase, uppercase, capitalize;
  • Pseudo-class is a keyword which is added to the selector to create styles for a specific state. e.g. ":hover" pseudo-class. defn: h3:hover{color:green}
  • position property (position:relative;) allows us to specify how CSS should move the element relative to its current position in the normal box model. Combined with offset values given through left, right, top, bottom it sets the position. e.g. position:relative; left:10px; creates left offset.
  • position:absolute; This locks the element in place relative to its parent container and the element is removed from notmal flow. Surrounding elements ignore it. Other offset (left, right, top, bottom) properties can be used as usual. Parent element must have a position rule (typically position:relative).
  • position:fixed; kind of absolute positioning. Locks the element relative to browser window. Other offset properties are available for use. The element is removed from normal flow. It is different from absolute positioning in that the element does not move when the user scrolls.
  • float:left or right usually pushes elements left or right. typically used with width property. width:50%. One can use comma to separate the element tags to asign same style to multiple elements. (h1, h2 {}) in css style.
  • z-index:higher makes the element move higher in the stack. Used when the elements overlap.
  • Center a block element horizontally. set margin:auto; or one can also do it by setting display:block; This works even for images.
  • color wheel is a tool that helps in visualizing color. Similar colors (hue) are neighbour and different colors are further apart. When two colors are opposite to each other they are called complimentary colors. When these xolors are placed side by side they create strong visual contrast. e.g. red (#FF0000) and cyan (#00FFFF). There are many color picking tools available online which can be used to pick colors.
  • Tertiary colors: RGB colors are called primary colors. Mixing two primary colors create secondary colors. G+B is cyan, R+B is magenta and R+G is yellow. Cyan is complement to R and opposite to R on the color wheel. Mixing primary color with neighbouring secondary color creates tertiary color. Split-complimentary color scheme is used to create tertiary colors. orange, cyan, raspberry etc are example of tertiary color. One page usually has dominant color and complimentary color is used to bring attention to certain content on the page. Nice blue color - teal - #09A7A1.
  • Hue, Saturation and lightness adjustment of color. Hue menas color . Saturation is the amount of gray in the color. Lightness means amount og black and white in the color. red - hsl(0,100%, 50%). Learn more about other colors. Mixing white creates tint and mixing black creates shade.
  • background:linear-gradient(gradient_direction, color1, color2, color3,...) creates gradient color distribution for the element. gradient_direction is in degree. 90deg means horizontal gradient and 45deg menas diagonal gradient. We can use repeating-linear-gradient() in-place of linear-gradient() to repeat a specified gradient pattern. Backgroung property also supports url() function to link to an image of chosen texture or pattern which is used as the background of the element.
  • transform property lets you scale, move, rotate, skew etc the element. When used with pseudoclass :hover it adds interactivity to the element. p:hover{transform:scale(2.2);} scales the paragraph to 2.2 when mouse is hovered over the element. Applying it over any div elemetn will afferct all child element.
  • transform:skewX(45deg) or transform:skewY(-32deg) skews element along X or Y axis by specified degree. CSS can be used to create interesting gtaphics like heart shape, crescent moon etc. ::before and ::after pseudo elemets are used in such things. These adds something before or after the selected element using a CSS style. These must have a defined content property. These contents might be text or image, empty strings etc.
  • animation properties and @keyframes rule control how animation works for an element. animation-name sets the name of the animation which is later used by @keyframes to tell which rule to use with which animation. animation-duration sets the length of time for the animation. @keyframes sets the frames for the animation duration. 0% sets the opening scene and 100% sets the ending scene. animation name and duration are set into the CSS declaration based on id of the element. check detail format if needed to use from www3school. One can also apply animation in the hover pseudoclass to animate the hover state property of an element. animation-fill-mode property specifies which style to apply to an element when animaion is finished. Movement of an element can be simulated with animation by applying rules to the position offset property when the position is fixed or relative.
  • One can create a visual direction from left to right using the opacity rule on an elemnt. Just fade the element from left to right by applying an animation on the element and setting the opacity value at defined keyframe.
  • Another animation property is animation-iteration-count which determines how many times one wants to loop through an animation. animation-iteration-count:infinite; sets the animation loop for infinte times. One can change the frame value (e.g. one may be set at 20% while other is at 50%) inside the @keyframes to change the rate of similar animation. Therefore multiple elements may be animated at variable rate.
  • animation-timing-function:ease (or ease-out, ease-in, linear) talks about the acceleration of the animation. One can also use cubic-bezier(0.25, 0.25, 0.75, 0.75) as a value. Learn more about it before use.

CSS flexbox

  • display:flex; is used to position containers inside a parent container.
  • flex-direction:row;row-reverse;column;column-reverse; aligns the children element horizontally or vertically
  • justify-content:center; flex-end;flex-start;space-between;space-around;space-evenly; These are the options to place the flex items inside the flex container along the main axis of the of the arrangement. The main axis of the arrangement is the direction along which the items are arranged.
  • align-items: flex-start;flex-end;center;stretch;baseline; These are the options to set the items along cross axis. For row as the main axis, these options are used to push the items in the entire row up or down within the container and the reverse is true for columns.
  • flex-wrap:nowrap(default);wrap;wrap-reverse; These options helps to wrap a flex item. Generally all the flex items are fit into a single flex container. If the option is used, the items are wrapped into new column or new row depending on the size of the items and the container.
  • flex-shrink:any number(1,2 etc). These property is applied to the flex items and not on the container. The items are shrink by this much factor when flex container is too small. Items shrink when the width of the parent container is smaller than the combined widths of all the flex items.
  • flex-grow controls the size of the items when the parent container expands: just the opposite of the flex-shrink which controls the behaviour when the parent container shrinks. It takes numbers as value just like flex-shrink.
  • flex-basis sets the initial size of the items before shrink or grow takes control. One can give %, px or rem as the argument for the value.
  • Above flex shrink, grow or basis can be applied all at once to the flex like, flex: 1 0 10px; which means grow:1, shrink:0 and basis:10px; default value is 0 1 auto;
  • Use order property to arrange the items inside the container. By default the values appeart in the order they come in html. order:1, 2 etc.
  • align-self:center etc. It is used to align each item individually because other CSS align properties do not work on the flex items.

CSS grid

  • display:grid; Using this any html element can be turned into a grid container. The parent element is referred to as container and its children are called items.
  • Create a container element. set its CSS display property to grid. Use grid-template-columns:50px 50px; to create two columns inside the container each 50px wide. Use grid-template-rows: 50px 50px ; to create two rows of 50px height.
  • Grid units can be used to change the size of the rows and columns. fr: sets the rows or the columns to a fraction of available space. auto:sets width or height automatically; %: sets to percent width of the conatiner. Example: grid-template-columns: auto 50px 10% 2fr 1fr;
  • Gap between columns: grid-column-gap: 10px; grid-row-gap:5px; grid-gap:10px 20px; sets 10px row gap and 20px column gap. Incase one value is given, it will create same gap between row and column.
  • To control the amount of columns an item will consume, one can use the grid-column:1/3; This will make the item to start at first vertical line of the grid on the left and span to the third line of the grid, which will consume two columns. similarly one can use grid-row.
  • In CSS grid, the content of each item is located inside a box which is referred as cell. One can align the content's position within its cell horizontally using justify-self:start or center or end. To align items vertically one can use align-self:center etc. If you want to align all the items horizontally, you can use justify-items:center, etc. If you want to align all the items vertically, you can use align-items:center, etc.
  • One can group the cells of a grid into an area and give the area a custom name. Use grid-template-ares: name of the columns; use '.' for an empty row. see more deatil. This is applied on the main container. In order to place an item in the custom area one can reference the name. .item1 { grid-area: header; } lets the grid know that you wanto to place the item1 class in the are named "header". grid-area can also be used without dreating any template and using only the line numbers. grid-are:hori start/vert start/ hori end/ vert end; grid-area:1/2/3/3;
  • Use repeat function to reduce repetition. grid-template-rows: repeat(100, 50px); creates 100 rows of 50px height each. one can also use multiple values inside repeat function. Similarly this can be used for rows.
  • use minmax(50px, 200px) to set max and minimum limit on the width.
  • repeat function has an option auto-fill which allows to automatically insert as many rows or colmumns of the desired size possible depending on the size of the container. auto-fit works exactly like auto-fill. It stretches items to fit to the size of the container. If all the items can not fit in a single row a new row is created.
  • Use media queries to rearrange grid areas, change dimensions of a grid and rearange placement of items.

Applied Accessibility

  • Use 'alt' attribute in images if necessary. If there are captions describing the images one can keep alt tag empty.
  • Use headings appropriately for different sections. Headigs should have semantic meaning. each page should have only h1 element which is the main subject of the page.
  • There are many tags to use to incorporate accessibility feature. e.g. main, header, footer, nav, article, and section. By default browsers views them similar as div element. Main element wraps the main content and there should be only one per page. This should not include content which are repeated in each page like navigation, banner etc. 'artcile' is a sectioning element and is used to wrap independent content. 'section' element groups related content. When there is no relation between groups of content use 'div'. 'header' wraps introductory information or navigation links for its parent tag or works for elements that are repeated in multiple pages.
  • 'nav' is used to wrap around the main navigational llinks in the page. 'footer' contains copyright information to links to related docs. 'audio' element is used to wrap sound or audio content. It supports the 'controls' attribute which adds functionality to play, pause etc. It does not need any value.
  • 'figure' and 'figurecaption' are used together to wrap chart in the page. The 'label' tag wraps text for a specific form control item. The 'for' attribute on the 'label' tag associates the label with id of the corresponding item. SInce radio buttons come in group often we can semantically show that the choices are part of a group by using 'fieldset' tag surrounding the radio button group. Use 'legend' tag alog with the 'fieldset' tag.
  • Add date picker. use type='date' in the input field to get a date picker. Use 'time' tag as an inline element to standarize time. This value will be accessed by assistive devices 'sup' is an superscript tag.
